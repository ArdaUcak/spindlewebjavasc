# STS – Spindle Takip Sistemi

Bu proje, spindle ve yedek parçaların CSV dosyaları üzerinden takip edilmesini sağlayan basit bir web uygulamasıdır.

## Gereksinimler
- Node.js 18+

## Çalıştırma
1. Gerekli dosyalar hazırsa sunucuyu başlatın:
   ```bash
   npm start
   ```
2. Tarayıcıdan `http://localhost:3000` adresine gidin.
3. Giriş bilgileri:
   - Kullanıcı Adı: `BAKIM`
   - Şifre: `MAXIME`

## Veri Dosyaları
- `spindle_data.csv`
- `yedek_data.csv`
- `takip_export.csv` (ihracat sonrası oluşur)
